This project focuses on analyzing real-world sales data using Python libraries such as Pandas and NumPy. The main objective is to uncover patterns and insights that can help in understanding business performance.

The dataset was first cleaned by handling missing values, removing duplicates, and converting data types. A new revenue column was created using numerical operations.

Further analysis was performed to identify monthly sales trends, top-selling products, highest revenue-generating countries, and most valuable customers. Aggregation techniques like groupby were used extensively to summarize the data.

This project demonstrates practical skills in data preprocessing, data analysis, and extracting actionable insights without relying on visualization tools.
